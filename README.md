## Ошибки при запуске веб-версии

Так как проект на ранней стадии разработки, технические сборки веб-версии происходят постоянно, поэтому пользователь может столкнуться с ошибками. Они появляются потому что данные игровых сессий хранятся в кеше браузера, из-за чего в дальнейшем они конфликтуют с новой версией игры. ПОЭТОМУ, либо чистите кеш данной веб-страницы, либо открывайте её в режиме инкогнито.
